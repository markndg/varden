Varden Web Shield

Connect to a local (or remote) Varden server and choose a protection mode. This is a one-time setup step.

First run

  1. Start Varden locally: varden web-shield demo (or point the endpoint above at your own server).
  2. Save settings here.
  3. Open the attack lab at <endpoint>/webshield/lab and click "Run all 20 cases".
  4. Watch the toolbar badge and the Web Shield dashboard update as tools are registered.
