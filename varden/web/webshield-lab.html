This page behaves like an ordinary website that exposes tools to a browser agent via document.modelContext.registerTool(). None of the 20 cases below perform any real destructive, financial, credential or exfiltration action — they only demonstrate what Varden Web Shield sees and how it reacts. Every registration and output below is sent to your local Varden server exactly the way a real integration (SDK or browser extension) would send it.

Varden Web Shield: runtime governance and tool-surface security for browser agents. This lab is a detection demonstration, not a sandbox and not a guarantee against every possible attack — see docs/web-shield-limitations.md in the repository for an honest accounting of what this can and cannot do.
